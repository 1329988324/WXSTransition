# WXSTransition
transition animation asset 

一行代码调用
```
 [self.navigationController wxs_pushViewController:[[SecondViewController alloc] init] animationType:<#(WXSTransitionAnimationType)#>];
```
包括系统转场动画在内，现在有40个转场动画
